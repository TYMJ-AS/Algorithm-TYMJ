## 해결된 코드
```python
from collections import deque

n = int(input())
q = deque(range(1, n + 1))

while len(q) > 1:
    q.popleft()           
    q.append(q.popleft()) 
print(q[0])

```

## 풀이
deque를 사용하여 문제를 해결하였다.
처음 deque에 1~n의 숫자를 담도록 구성하고
while문을 통해 deque의 길이가 하나만 남도록 루트를 돌렸다.
popleft를 사용하여, 제일 왼쪽에 있는 요소를 제거하고,
q.append(q.popleft())를 수행하면서, 제일 왼쪽에 있는 요소를 또 제거를 수행하면서, 가장 오른쪽에 삽입하였다.
이러한 과정을 반복하면 원하는 형태의 값이 출력된다.
