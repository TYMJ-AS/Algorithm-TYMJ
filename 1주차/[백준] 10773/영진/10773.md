## 해결된 코드
```python
import sys
input = sys.stdin.readline
a = int(input())

stack = []
for i in range(a):
    b = input()
    if int(b) == 0:
        if len(stack) > 0:
            stack.pop()
    else:
        stack.append(int(b))
if len(stack) == '0':
    print(0)
else:
    print(sum(stack))
```


## 풀이
a를 인자로 받아, a만큼 반복문을 수행하여, b만큼 입력을 받았다.
b가 만약 0이라면 스택이 비어있지 않을때 pop을 수행하여, 맨 뒤에 들어오는 값을 제거했다.
아닐 경우 b를 stack에 쌓이도록 하였다.
최종적으로 스택이 비어있을 경우 0을 반환하고, 쌓여있을 경우 합을 반환하여 문제를 해결하였다.
