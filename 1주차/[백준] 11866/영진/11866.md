## 해결된 코드
```python
a, b = map(int, input().split())

box = list(range(1, a + 1))
new_box = []
idx = 0

while box:
    idx = (idx + b - 1) % len(box) 
    new_box.append(box.pop(idx))

print("<" + ", ".join(map(str, new_box)) + ">")
```

## 풀이
처음 인자값을 리스트에 담고, box라는 리스트를 생성하여, 1~a까지의 숫자를 담아냈다.
이후 박스에 요소가 존재하지 않을때까지 while문을 돌렸다.
인덱스 범위에 벗어나지 않도록 idx를 설계하고, idx위치에 pop이 수행되면서, 새로운 리스트에 삽입을 수행했다.
문제에서 요구하는 출력형식과 맞게 ", "을 join을 활용하여 삽입을 수행하였다.
